# homebrew-monopam

Homebrew tap for OCaml tools.

## Installation

```bash
brew tap samoht/tap
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| `cascade` | CSS generation and manipulation tool |

## Usage

```bash
# Install pre-built binaries
brew install cascade

# Or build from source
brew install --HEAD cascade
```
